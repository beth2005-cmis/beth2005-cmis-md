How to create a new Website

  1. Clone into your repository
  2. gedit a file called index.html, and put appropriate codes
  3. Validate code
  4. gedit a file called main.css, and put appropriate codes for decoration
  5. Validate code
  6. Start the development server using python -m SimpleHTTPServer
  7. Type localhost:8000 to check
  8. Validate code
  9. git add/commit/push to save your work in your repository
  10. Start the production server- sftp username@cmis-hsteacher
  11. Change the directory of the sftp using cd public_html.
  12. then, put index.html and main.css separately
